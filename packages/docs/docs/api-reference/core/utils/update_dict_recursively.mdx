## <kbd>function</kbd> `update_dict_recursively`

```python
update_dict_recursively(old_object: Any, new_object: Any) â†’ Any
```

Update a given dict recursively with data from a new dict. 

It will not add any properties and assert that the types remain the same (or null). null->int or int->null is possible but not int->dict or list->int. 



**example:**
 ```python
merge_dicts(
     old_object={"a": 3, "b": {"c": 50, "e": None}},
     new_object={"b": {"e": 80}},
) == {"a": 3, "b": {"c": 50, "e": 80}}
```
