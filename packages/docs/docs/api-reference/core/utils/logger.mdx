## <kbd>class</kbd> `Logger`
A reimplementation of the logging module. 

_Why?_ The `logging` module from the Python standard library is kind of hard to configure with some setups we have on the server-side. That is why we have decided to use a custom logging class in some of our bigger projects instead. 

This class is simply formatting log lines, appending them to the log files and providing a simple API.

### <kbd>method</kbd> `archive`

```python
archive(keep_last_hour: bool = False) → None
```

Move all log lines in "logs/info.log" and "logs/debug.log" into an archive file ("logs/archive/YYYYMMDD-debug.log", "...info.log"). 

With keep_last_hour = True, log lines less than an hour old will remain.
