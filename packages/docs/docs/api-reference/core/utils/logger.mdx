## <kbd>function</kbd> `log_line_has_time`

```python
log_line_has_time(log_line: str) → bool
```

Returns true when a give log line (string) starts with a valid date. This is not true for exception tracebacks. This log line time is used to determine which file to archive logs lines into.

### <kbd>method</kbd> `archive`

```python
archive(keep_last_hour: bool = False) → None
```

Move all log lines in "logs/info.log" and "logs/debug.log" into an archive file ("logs/archive/YYYYMMDD-debug.log", "...info.log"). 

With keep_last_hour = True, log lines less than an hour old will remain.
