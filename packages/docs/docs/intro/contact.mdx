## Developers

## Bug Reports & Feature Requests

You can track things we are actively working on GitHub: https://github.com/tum-esm/pyra/issues. We gladly accept bug reports and feedback on Pyra. Feel free to ask questions about possible features or discuss similar research endeavours.

Planned updates are listed as [milestones on GitHub](https://github.com/tum-esm/pyra/milestones).
